# Data Set Description

Training data comprises **5.2 million** SKU-day price observations from the ecommerce platform (2022–2024), enriched with competitor pricing scraped daily.

| Category | SKU-Days | Avg. Elasticity |
| --- | --- | --- |
| Electronics | 1,800,000 | -1.4 |
| Fashion | 1,600,000 | -2.1 |
| Home & Garden | 1,200,000 | -0.9 |
| FMCG | 600,000 | -0.5 |
